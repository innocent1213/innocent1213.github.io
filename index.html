情书

发表于 2022-12-05 更新于 2022-12-06 分类于 电影观后感

这部爱情片，是打破常规的爱情片，他通篇没有男主的直接描述，从男主生命中两个爱人的回忆视角，描绘了不同年龄阶段、不同性格的女主的两段截然不同的爱恋。

“你还好吗？我很好！”

渡边博子的痴情让人感动，也让人羡慕。情书的主线可以说是渡边博子对于藤井树放下的过程。在藤井树去世两年后，依然找寻着他的痕迹，给他寄信，收到回信后，不假思索地欺骗自己他还在看着她。在秋叶打破她的幻想的时候，她表现出的心碎，无不表现出她的痴情。但是逝者已逝，生者还有自己的生活，还有眼前人，活在悲痛中，只会徒增烦恼，失去更多，比如秋叶，也因为她的表现伤透了心。但即使这样，秋叶依旧带着博子前往回信人的家里，前往藤井树遇难的山上，帮助博子解开心结，迎向新生。虽然一开始对于秋叶并无好感，但是随着剧情的推进，越发地喜欢他和佩服他。没有他，博子可能要一直沉浸在回忆中、伤痛中无法自拔，无法向前看。

发表于 2022-11-30 分类于 思考人生

近况记录

在写这个想法之前，还是想先理一下近期状态。在Azla打工的过程中，对于未来越来越迷茫，不知道建筑还能不能继续下去，不知道我的建筑学水平能不能找到一份好的工作，不知道我需不需要转行做一些其他的未知的领域。同门和朋友的转行热情也进一步地让我思考和迷茫。目前只能没有目的性地学习一些乱七八糟的东西（python），但也并没有很深入的学习。

虽然我认为，在迷茫的时候应该多看点书，但是又被一些快餐娱乐所吸引，根本沉不下心来，所以一直未开始。一些美其名曰热爱生活的事情，现在想想只不过是逃避现实，陷于玩乐的借口罢了，令人羞愧。

开篇

发表于 2022-11-22 更新于 2022-11-30

一切的开始

我为什么想写自己的博客？起因是偶然间看到别人写的博客，记录自己的想法，自己的生活，自己的小作品，觉得很有意思，感觉博客是对自己生活、自己想法的记录。同时博客是自己搭建的，记录和运营他也有成就感，让我有继续下去的动力。同时也觉得自己活得有些浑浑噩噩，虽然经常自诩有自己的想法，但是没有记录的想法很多时候不会存在太久，最终变成一片混沌，止步不前。

这个博客是我自己搭建，由我自己完善、优化，也将由我自己填充、丰富，不出意外的话，也会只有我自己看到，这些话、这些文字都是写给我自己的，不需要太多的约束，这是记录我内心、我的生活的博客。在未来，希望能写一些记录生活、记录我想法的文章，以及一些学习心得、读书笔记、观后感等等，希望通过这个媒介，对自我进行提高，逐渐明白我自己到底是怎样，到底想做什么。

希望这个博客是个开始，是记录的开始，热爱生活的开始，改变自己的开始。特此记录，希望能够不忘本心。
